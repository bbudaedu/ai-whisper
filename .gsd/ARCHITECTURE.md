## Components

### Backend API (`api_server.py`)
- **Purpose:** 提供儀表板後端，管理播放清單設定、手動觸發轉錄任務、即時日誌串流與狀態監控。
- **Location:** `api_server.py`
- **Dependencies:** `fastapi`, `uvicorn`, `pydantic`, `pipeline.playlist_manager`.

### YouTube Automation (`auto_youtube_whisper.py`)
- **Purpose:** 主程式循環，偵測新影片、使用 `yt-dlp` 下載並調用 Whisper 進行轉錄。
- **Location:** `auto_youtube_whisper.py`
- **Dependencies:** `yt-dlp`, `faster-whisper`, `gpu_lock`.

### AI Proofreading (`auto_proofread.py`)
- **Purpose:** 將 Whisper 生成的 SRT 段落發送至 Gemini API 進行詞彙校對與斷句修正。支持讀取代書 PDF 以輔助校對佛學專有名詞。
- **Location:** `auto_proofread.py`
- **Dependencies:** `google-genai`, `pdfplumber`.

### Post-Processing (`auto_postprocess.py`)
- **Purpose:** 整合校對後的文字，進行對齊、標點符號處理，並生成 `.xlsx` 與 `.docx` 報表文件。
- **Location:** `auto_postprocess.py`
- **Dependencies:** `openpyxl`, `python-docx`, `pandas`.

### Resource Management (`gpu_lock.py`)
- **Purpose:** 使用系統鎖確保同一時間只有一個重負荷的 GPU 任務執行，防止顯存溢出。
- **Location:** `gpu_lock.py`

## Data Flow

1. **偵測階段**: `auto_youtube_whisper.py` 透過 `PlaylistManager` 讀取 `config.json` 中的播放清單，並與 `processed_videos.json` 比對，找出尚未處理或不完整的影片。
2. **獲取階段**: 下載影片音檔並轉化為 16kHz WAV。
3. **處理階段**:
    - `faster-whisper` 生成原始 `index.srt` 與 `index.txt`。
    - `Gemini` 根據參考 PDF 進行文本校對，生成 `_proofread.srt`。
4. **生成階段**: 整合兩者，透過 `openpyxl` 生成多分頁 Excel 供人工校對。
5. **通知階段**: 將成品上傳 NAS 並發送 Email 通知。
