# UI-SPEC

## Spacing
- Spacing scale：4/8/16/24/32/48/64
- Touch target 最小 44px
- Icon-only buttons 仍需保留約 44px 點擊區域

## Typography
- 字級：14/16/20/24 px
- 字重：400/600
- 行高：body 1.5、heading 1.2

## Color
- Background：light #f9f9fb、dark #0f1115
- Secondary：light #ffffff、dark #161b22
- Accent：#3B82F6
- Destructive：#ef4444
- Accent 使用情境：primary CTA、active nav/tab、primary progress/badge、primary links

## Copywriting
- Primary CTA label：「提交任務」
- Destructive 確認：delete playlist 需要 confirm dialog（不需 keyword）、disable tracking 需要 confirm、cancel task 需要 confirm
- Empty/error copy：接受預設文案
