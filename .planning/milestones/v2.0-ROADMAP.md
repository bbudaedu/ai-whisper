# Milestone Archive: v2.0 (2026-03-29)

**全系統自動化 E2E 測試與穩定性增強**

## 🎯 Goals
建立一套涵蓋 legacy 功能與 v1.0 新功能的自動化測試框架，確保全系統（API, Pipeline, Web UI）在持續疊代中的穩定性。

## 📋 Scoping Results
- **測試框架**: 採用 Pytest (Backend) 與 Playwright (Frontend) 雙層體系。
- **環境模擬**: 使用 Mock Executor 與 in-memory SQLite 進行隔離測試。
- **測試資料**: 準備了 `tests/fixtures/v2/sample.wav` 作為基準音訊。
- **覆蓋範圍**: 完整覆蓋了 External Web UI 與核心 API。

## 🏆 Accomplishments
- **全自動化測試基礎設施**: 建立了專案首個完整的雙層測試體系（Backend Pytest + Frontend Playwright）。
- **GPU 資源並發保護**: 實作並驗證了基於 `flock` 的跨進程 GPU 互斥鎖。
- **講者名稱 (Speaker Name) 端到端編輯**: 完成了從資料庫、API 到 Web UI 的完整功能鏈。
- **AI 校對精準度提升**: 在 LLM Prompt 中引入 `speaker_name` 注入與自定義佔位符支援。
- **響應式 UI 驗證**: 透過 Playwright 行動版模擬器確保行動裝置操作體驗。
