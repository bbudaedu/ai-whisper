# Milestone Archive: v1.0

**Archived:** 2026-03-24
**Status:** Completed
**Tag:** `v1.0`

## Summary

FaYin v1.0 成功建立了語音處理平台的基礎設施，將 Whisper 語音轉錄能力 API 化，並提供了 Mobile-first 的外部 Web UI。系統具備穩定的任務佇列管理、多階段處理管線（下載、聽打、校對、排版）、安全認證（JWT + Google OAuth）以及長期歷史存檔功能。

## Key Accomplishments

1. **[架構] 統一任務佇列與 GPU 排程**: 支援單 GPU 資源下的異步任務處理，具備優先級與自動重試機制。
2. **[API] 安全外部介面**: 實作了基於 JWT 的 API 認證，提供任務建立、狀態查詢與結果下載端點。
3. **[UI] Mobile-first 客戶端**: 獨立的外部 Web UI，支援 Google OAuth 登入、檔案上傳與即時任務追蹤。
4. **[功能] YouTube 自動監控**: 實作播放清單追蹤邏輯，自動偵測新集數並排入處理流程。
5. **[後製] 校對與說話者標註**: 建立了 LLM 校對管線與 Pyannote 說話者分離架構，支援標註 Speaker A/B/C/D。
6. **[持久化] 長期歷史查詢**: 統一資料庫儲存，任務紀錄與產出檔案永久保留，並提供 UI 檢索。

## Phase Details

### Phase 1: 任務佇列與排程基礎
- **Goal**: 任務佇列化與分段處理，單 GPU 排程。
- **Outcome**: 成功實作 `TaskScheduler` 與 `TaskRepository`，確保內部任務優先。

### Phase 2: 對外 API 與認證 (含 02.1)
- **Goal**: 提供安全 API 與使用者登入。
- **Outcome**: 實作 JWT 認證、Email/Password 登入與 Google OAuth。

### Phase 3: 外部 Web UI 與提交流程
- **Goal**: Mobile-first 提交與追蹤介面。
- **Outcome**: 交付 React-based `web-ui-external`，支援多格式下載與通知設定。

### Phase 4: 校對增強與說話者標註
- **Goal**: 提升轉錄品質與標註說話者。
- **Outcome**: 整合 `pyannote.audio` 標註 A/B/C/D，並預留 LLM 校對管線。

### Phase 5: 長期保存與歷史查詢
- **Goal**: 任務與檔案永久保留。
- **Outcome**: 實作歷史查詢 API 與前端頁面，支援過往產出檔案隨時下載。

### Phase 6: 里程碑缺口修復
- **Goal**: 解決整合問題與文件同步。
- **Outcome**: 修正 API 路徑不一致與格式映射問題，完成全系統驗證。
