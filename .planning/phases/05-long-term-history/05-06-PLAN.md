    2. 在 download_task_results 函式簽名新增參數：
       - `format: str | None = Query(default=None, description="Optional format filter: txt,srt,vtt,json,tsv,docx,xlsx")`

    3. 在取得 files 後、建立 zip 前加入 format 過濾邏輯：

       - 以 requested_suffix 過濾 `files = [path for path in files if path.suffix.lower() == requested_suffix]`。
       - 若過濾後 files 為空，回傳 404：`detail="Requested format not found"`。
    4. zip 檔仍維持輸出，但僅包含過濾後的檔案；zip_name 保持 `{task.id}_{timestamp}.zip`。
